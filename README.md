## Installation

Install R package `devtools` from CRAN:

``` r
install.packages("devtools")
```

Install R package `hfmisc`:

``` r
devtools::install_github("hfmisc")
```

Load package:

``` r
library(hfmisc)
```

Get overview of functions:

``` r
help(package = "hfmisc")
```
